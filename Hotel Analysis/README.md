# Hotel Booking Analysis Dashboard

A comprehensive Power BI dashboard suite to analyze hotel booking trends, customer behavior, occupancy, distribution channels, and waitlists. This project helped us sharpen our skills in **data modeling**, **DAX**, and **interactive dashboard design**.

---

## üîç Project Overview

Using actual hotel booking data, this project aims to extract meaningful business insights to support decision-making in hotel management.  
We built **5 fully interactive dashboards** that answer key operational and marketing questions for hotel managers, analysts, and business stakeholders.

---

## üìä Key Features

- **Booking Overview Dashboard**  

- **Customer Dashboard**  

- **Occupancy Dashboard**  

- **Distribution Dashboard**  

- **Waiting List Analysis Dashboard**  


---

## üóÑ Data Sources

- **Source:** Publicly available hotel booking dataset from [Kaggle](https://www.kaggle.com/)  
- **Records:** ~119,000  
- **Columns:** Booking ID, Customer Type, Hotel Type, Market Segment, Distribution Channel, Arrival Date, Booking Status, Revenue, Room Type, Special Requests, and others  
- **Transformations:** Data cleaning, missing value handling, type conversions, and calculated columns for DAX measures

---

## üõ† Tools & Technologies

- **Business Intelligence:** Power BI, Power Query, DAX  
- **Data Modeling:** Star Schema, Relationships, Parameter Inputs  
- **Design & Interactivity:** Interactive dashboards, bookmarks, tooltips

## üìä Data Model Description

- **Tables:** Bookings, Customers, Rooms, Revenue, Calendar  
- **Relationships:**  
  - Star Schema design with Bookings as the central fact table  
- **Modeling Approach:** Optimized for performance and interactive filtering, enabling slicers and drill-through between dashboards

## üß≠ How to Use

1. Download the PBIX file from this repository.  
2. Open it in **Power BI Desktop (latest version recommended)**.  
3. Navigate between dashboards using bookmarks or tabs to explore insights.  
4. Hover over visuals to see tooltips with additional metrics.
